# Usage

require Node.js v18.20.8 or v20.3.0, v22.0.0 or higher. 

## 1. clone the repo

```
git clone https://github.com/221934420a/joeffrey-wong_astro-blog.git
```
## 2. move to the folder

```
cd ".\joeffrey-wong_astro-blog"
```

## 3. install dependencies

```
npm install
```

## 4. run dev server

```
npm run dev
```
